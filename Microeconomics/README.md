## Folder Structure

The microeconomic year can be categorized, roughly, into 6 topics:

1. Partial Equilibrium: Microeconomic Theory Fundamentals, Market Power, Quasilinear Economy
2. Decision Theory: Expected Utility Theory, Risk Aversion
3. Static Games: Nash Equilibrium, Repeated Games
4. General Equilibrium: Welfare Theorems, Auctioneer/Walrasian GE, Edgeworthian GE (the core), 
5. Bayesian Games: Equilbrium Refinements, Auctions, Signaling, Screening, Principal-Agent
6. Mechanism Design: IPVM, Revenue Equivalence, Symmetric BNE

This folder contains a subfolder for each to preserve a clean separation of content. In addition, I include homeworks and old exams to practice. The SolutionBank and Tools are meant to be live documents. The former recording a log of exercises I have solved with a representative name to help recall; while the latter records mathematical tools useful for solving PhD microeconomics problems.
